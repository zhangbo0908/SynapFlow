# SynapFlow Design Assets

## Logo Design Concept (Scheme C: Synaptic Flow)

The SynapFlow logo embodies the core concepts of "Synapse" (connection) and "Flow" (fluidity).

### Visual Metaphor
- **The Curve**: An abstract 'S' shape that flows from bottom-left to top-right, representing the journey of thought and the fluid nature of mind mapping.
- **The Branch**: A horizontal extension forming an abstract 'F', symbolizing the branching capability of neural networks and mind maps.
- **The Nodes**: Circular terminals representing data points, ideas, or synaptic junctions.

### Color Palette

The gradient represents the transition from logic (Blue/Cyan) to creativity (Purple).

| Color Name | Hex Code | Tailwind Class | Usage |
|------------|----------|----------------|-------|
| Cyan 400   | `#22d3ee`| `text-cyan-400`| Start of flow, logic, clarity |
| Indigo 400 | `#818cf8`| `text-indigo-400`| Transition, depth |
| Purple 400 | `#c084fc`| `text-purple-400`| End of flow, creativity, spark |
| Slate 800  | `#1e293b`| `bg-slate-800` | Background gradient start |
| Slate 900  | `#0f172a`| `bg-slate-900` | Background gradient end |

### Files
- `logo.svg`: The master vector file. Scalable to any size.

### Usage
- Use the SVG directly in React components or `<img>` tags.
- For Electron app icons, convert this SVG to `.icns` (macOS) and `.ico` (Windows).
